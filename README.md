# DentalCockpit Pro

A comprehensive dental practice management system with AI-powered features.

## Features

- **Patient Dashboard** - Complete patient information with real-time data synchronization
- **AI-Powered Analysis** - Automatic radiographic analysis and diagnostics
- **Voice Recording & Transcription** - Record consultations and generate anamnesis automatically
- **X-Ray Viewer** - DICOM-compatible radiographic viewer with AI assistance
- **INAMI Integration** - Belgian healthcare system integration for e-Health services
- **Certificate Generation** - Automated medical certificates
- **Mutuelle Verification** - Insurance verification and coverage checks
- **Timeline & History** - Complete patient history tracking

## Live Demo

Visit the live application: [DentalCockpit Pro](https://ismaikami.github.io/K2-Dent/)

Single-page HTML/CSS/JavaScript application with modern dark theme UI optimized for dental practice workflow.
